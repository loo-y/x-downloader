# x-downloader v0.2.0

发布日期：2026-04-12

## Highlights

- 新增 YouTube 下载支持，`xdl` 现在可统一处理 X/Twitter 与 YouTube 链接
- 支持保存默认下载目录与默认 cookies 文件路径，减少重复输入
- 默认自动探测 Chrome profile，并优先尝试带 X 登录态的 profile
- 支持下载后直接调用 `ffmpeg` 做片段裁切
- 增加最小自动化回归测试，覆盖 URL 校验与 CLI 帮助文案

## User-facing changes

- 现在支持以下视频链接：
  - `x.com`
  - `twitter.com`
  - `youtube.com`
  - `youtu.be`
- `xdl --help` 的说明已经同步为 X/Twitter + YouTube
- CLI 仍默认忽略环境代理，除非显式传入 `--proxy` 或 `--use-env-proxy`

## Notes

- YouTube 真实下载已做人工验证
- 自动化测试当前主要覆盖：
  - URL 校验
  - 缺少 URL 时的错误提示
  - CLI 帮助文案
- 真实下载链路仍主要依赖 `yt-dlp` 的上游兼容性

## Known limitations

- 还没有完整的自动化集成测试来覆盖真实下载流程
- YouTube 的 age-restricted / 会员内容等特殊场景仍完全依赖 `yt-dlp` 默认行为
- Chrome profile 自动探测逻辑跨平台已实现，但不同机器环境仍建议首轮实机确认
